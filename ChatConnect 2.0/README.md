To download Node Modules

For Backend

```shell
cd server
yarn
```

For Frontend.
Open another terminal in folder.

```shell
cd FrontEnd
yarn
```

For Frontend.

```shell
cd FrontEnd
yarn start
```

For Backend.

Open another terminal in folder, Also make sure mongodb is running in background.

```shell
cd server
yarn start
```

Done! Now open localhost:3000 in your browser.
